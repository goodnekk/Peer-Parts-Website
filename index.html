Peer Parts

Peer Parts is a platform for open design that allows you to create and share components for products, machines and systems.

Parts that fit together are exchangeable, making it easy to share, (re)use, (re)mix and (re)combine parts between projects.

The main goal of the Peer Parts project is to develop an open-source file format and a free CAD environment. It is currently in development.
